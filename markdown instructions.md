# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом, необходимо обрамить его (*)   
Например вот так *"текст"*


Чтобы выделить текст полужирным, необходимо обрамить его (**)   
Например вот так **"текст"**
